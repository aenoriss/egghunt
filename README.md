# Egghunt

AR campus easter-egg hunt — compete to find augmented-reality eggs, watch the game live from a digital twin.

## Structure
- `frontend/` — the AR.js / WebXR client (marker-based egg hunting)
- `backend/` — Express + Sequelize game server (players, eggs, real-time state)

*Consolidated from the former `eggFront` + `eggBack` repos (2026-07-05); full history preserved.*
